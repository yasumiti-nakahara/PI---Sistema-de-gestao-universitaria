# PI---Sistema-de-gestao-universitaria
Projeto universitário visando a criação de um sistema que tem como funções o cadastro de usuários e consulta de seus acessos.
O sistema é embasado no sistema proposto na atividade, onde tivemos como referência a primeira entrega realiada, sendo composto pela interface de utiliação onde temos os cadastros de pessoa física, Jurídica, cadastro de aluno, professor e por fim o cadastro de fornecedores.
O projeto possibilita a consulta do cadastro do usuário e caso não tenha cadastro ele possibilita o cadastro dentro das possibilidades citadas anteriormente
