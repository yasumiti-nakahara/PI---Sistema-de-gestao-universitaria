# PI — Sistema de Gestão Universitária

Sistema universitário para gestão de usuários, cadastro e consulta de acessos (alunos, professores, fornecedores, etc.). Esse projeto foi desenvolvido como parte de um trabalho de programa­ção universitária, com frontend, backend e persistência de dados.

---

## Sobre o Projeto

O objetivo deste sistema é fornecer uma aplicação para:

- Cadastro de **pessoas físicas** (alunos, professores) e **pessoas jurídicas** (fornecedores).  
- Diferentes perfis de usuário: aluno, professor, fornecedor, administrador.  
- Consulta de registros de usuários já cadastrados.  
- Possibilidade de registrar novos usuários diretamente pelo sistema, com validação e controle de acesso.  
- Interface web (frontend) para interação com os módulos do sistema.  
- Persistência de dados por meio de banco de dados relacional.

Este sistema serve como uma base para uma aplicação mais ampla de gestão universitária, podendo evoluir para incluir funcionalidades como controle de matrículas, notas, frequência, e outras.

---

## Tecnologias Utilizadas

A seguir, as principais tecnologias previstas (ou utilizadas) no projeto:

- **Backend**: Java  
- **Frontend**: HTML5, JavaScript  
- **Banco de Dados**: MySQL 
- **Controle de versão**: Git / GitHub
